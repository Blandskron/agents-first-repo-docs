Prompt Maestro Universal

Copiar, pegar y convertir cualquier repo

Un prompt maestro universal para que cualquier agente audite, ordene y reconstruya la documentación de un proyecto sin asumir stack, herramienta ni arquitectura.

Flujo Simple

Copiar -> Pegar -> Auditar -> Aprobar -> Aplicar. Sin onboarding largo, sin framework y sin estudiar una metodología completa.

Seguro por Diseño

El agente debe leer primero, pedir aprobación antes de cambios sensibles y evitar secretos, comandos inventados o arquitectura ficticia.

Universal

Sirve para Claude, Gemini, Copilot, ChatGPT, Codex o cualquier agente moderno, en repositorios nuevos, heredados o multi-stack.
